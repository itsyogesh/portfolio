Bitcharge is a crypto payments platform that lets businesses accept cryptocurrency payments and tips.

## Version 1 (2018-2019)

The first version was built with React and shipped to 195 users at bitcharge.co. It served as a B2B crypto payment acceptance platform.

## Version 2 (In Progress)

Rebuilding from scratch with a modern stack: Next.js 15, React 19, Fastify 5, Prisma, and PostgreSQL. Web3 integration through RainbowKit and Wagmi for a much better UX.

## Coinbox

Alongside Bitcharge, I'm building Coinbox — a local-first crypto portfolio tracker using Tauri 2.0, SQLite with encryption, and the Bitcoin Development Kit. Privacy-first, no server dependency.
